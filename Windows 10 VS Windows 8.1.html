Windows 10 Vs Windows 8.1

Design

For design I think the Windows 8 start screen is better than the Windows 10 start menu because the Windows 8 start screen is more colourful than the windows 10 start menu. Also I think the layout of the start screen is better.

Features

For Features I think Windows 10 wins because it has a lot of new features like Cortana, The new voice assistant in Windows 10, or being able to have multiple desktops open at one time. There is also a new action centre in Windows 10. (The action centre is a place where it shows your notifications.)

Overall

If you are not a fan of the Windows 8 start screen then upgrade to Windows 10. If you don’t want any new features stay on Windows 8.
